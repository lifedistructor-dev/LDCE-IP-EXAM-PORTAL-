Key Definitions (Ch. II) 🔴महत्वपूर्ण परिभाषाएँ
Dak / डाक
All communication received/issued by an office.
Case / केस
A current file or receipt with related papers.
Current File
File on which action is not yet complete.
Note / नोट
Remarks recorded to facilitate examination & decision.
PUC
Paper Under Consideration — the receipt being examined.
FR
Fresh Receipt — subsequent receipt bringing new info. Multiple = FRI, FRII...
Diary Number
Serial no. in Section diary + code (H=Hindi, O=Other) + year + section symbol.
Docketing
Entries in notes portion about serial no. assigned to each correspondence item.
Single File System (SFS)
Non-Secretariat Office refers file directly to Department for sanction/order.
DRO
Departmental Record Officer — nominated under Public Records Act 1993.
Desk Officer
A Section Officer designated to exercise delegated powers (e.g. financial sanctions).
Night Duty Clerk
Functionary handling dak outside office hours.

File Management & NOTING — Full Guide 🔴फाइल प्रबंधन एवं टिप्पणी — पूर्ण मार्गदर्शिका
Rule 13-14: Process Sheets & Disposal Level

Standard Process Sheets for repetitive cases (leave, GPF advance, HBA) — no notes required.

Max 4 levels for decision. If levels jumped, on return the file passes through ALL jumped levels for reconsideration.

Rule 15: Guidelines for NOTING 🔴 Most Asked
  • Notes must be concise & to the point; extra material → Appendix
  • Black or Blue ink only for handwritten notes (all staff/officers)
  • Dealing Hand: full signature + name + date on LEFT, below note
  • Officer: full signature on RIGHT + name + designation + date
  • Electronic environment → Digital signature at each level
Rule 16: Modification of Notes/Orders
  • Senior officer must NOT alter/replace a junior's note — must record own fresh note
  • Pasting over a note to conceal is PROHIBITED
  • Withdrawal of a mistaken decision needs approval of an officer senior to the one who took the original decision
Rule 19-22: Oral Instructions
SituationRule
Oral discussionsRecord conclusions on file at earliest; get confirmed by participants
Oral instruction (no time to write)Act, then follow up with written confirmation
Minister's PS conveys oral orderMust confirm in writing immediately
Urgent case, Minister on tourPrivate Secretary conveys decision; confirmation obtained later
Rule 23-24: Filing & Arrangement of Papers
  • Papers punched on left top corner, tagged right-to-left chronologically
  • Notes portion tag → left side of cover; Correspondence portion tag → right side
  • File exceeds 150-200 pages → mark "Volume I", continue as "Volume II"...
📌 Arrangement order (top→down): Reference books → Notes (ending with note for consideration) → Running summary → Draft for approval → Correspondence → Appendices → Standing guard file → Other papers → Recorded files → Routine notes
Rule 25: Referencing 🔴
  • Each part (Notes / Correspondence / Appendices) numbered separately, top-right corner
  • Document under consideration flagged as "PUC"
  • Latest fresh receipt flagged as "FR" (multiple → FRI, FRII...)
Rule 26-27: Linking & Urgency Grading 🔴

Inter-connected files → linked; de-linked after action, taking extracts.

GradingUse
ImmediatePrompt attention required
PriorityPrecedence over ordinary cases
Top PriorityExtremely urgent cases

Forms of Communication — FULL FORMAT 🔴पत्राचार के प्रारूप — संपूर्ण
🔴 Exam Favourite: Identifying WHICH form to use for WHICH purpose, and matching salutation/complimentary close.
1. Letter🔴
Used for: State Govts, constitutional bodies, attached/subordinate office heads, PSUs, public, statutory authorities. Salutation: "Sir/Madam"
No.... Government of India Department of... New Delhi, the...(Date) To Subject: Sir/Madam, With reference to your letter No...dated... I am directed... Yours faithfully, (A.B.C.) Under Secretary to the Govt. of India
2. D.O. (Demi-Official) Letter🔴
Used for: correspondence between officers to invite personal attention; personal & friendly tone; written in first person. Officer writing should ordinarily not be more than 1-2 levels below addressee.
D.O.No... GOVERNMENT OF INDIA Department of... New Delhi, the... My dear/Dear Shri... We propose to draw up... With regards, Yours sincerely, (XYZ) Deputy Secretary
3. Office Memorandum (O.M.)🔴
Used for: corresponding with other Departments / calling info from or conveying info to employees; also attached/subordinate offices. Written in third person, no salutation, no complimentary close.
No.... OFFICE MEMORANDUM Subject: The undersigned is directed to refer to... 2. Doubts have been expressed whether... (A.B.C.) Under Secretary to the Govt. of India
4. Inter-Departmental Note (I.D. Note)🟠
Earlier called U.O. Note (before 7th Edition CSMOP). Used for obtaining advice/views/concurrence of other Depts. Can be recorded on file OR as independent self-contained note.
Department of... Subject: 1. The present rules regulating... 2. A question has now arisen whether... 5. This department will be grateful for advice... (X.Y.Z.) Deputy Secretary Department of Legal Affairs — I.D. No...Dated...
5. Office Order🟠
Used for: internal administration — leave sanction, distribution of work, appointments, transfers.
No.... OFFICE ORDER (No. /2014) Shri X.Y.Z., Section Officer, is granted earned leave for...days from...to... (A.B.C.) Under Secretary to the Govt. of India Copy to: Office order file / Cashier / Section
6. Order🟠
Used for: certain financial sanctions and communicating disciplinary orders.
No.... ORDER Sanction of the President is accorded under rule 10 of the Delegation of Financial Powers Rules, to write off irrecoverable loss of Rs... (A.B.C.) Under Secretary to the Govt. of India
7. Notification🟠
Used for: notifying statutory rules/orders, appointments & promotions of gazetted officers via Gazette of India.
[To be published in the Gazette of India, Part I, Section 2] NOTIFICATION No... Shri XYZ, Under Secretary, is appointed to officiate as Deputy Secretary vice Shri... (A.B.C.) Joint Secretary To: The Manager, Govt. of India Press, Faridabad
8. Resolution🟡
Used for: public announcement of important policy decisions — e.g. industrial licensing policy, appointment of committees/commissions of inquiry. Also published in Gazette (Part I, Section 1).
RESOLUTION The Government of India have had under consideration the question of... (A.B.C.) Secretary to the Government of India ORDER ORDERED that a copy be communicated to... ORDERED that the resolution be published...
9. Press Communiqué / Note🟡
Used when giving wide publicity to a decision. Communiqué = more formal, reproduced intact. Press Note = hand-out, press may edit/compress.
Not to be published before... PRESS COMMUNIQUE/NOTE In response to public demand, the Government has appointed a Commission to go into... Forwarded to Principal Information Officer, PIB (A.B.C.) Joint Secretary
10. Endorsement🟢
Used when a receipt returned in original / copy sent to another Dept for info/action. Not used for communicating copies to State Govts (use Letter form there).
No.... A copy each of the papers mentioned below is forwarded for information and necessary action. (A.B.C.) Under Secretary to the Govt. of India List of papers forwarded: 1. 2.
11. Minutes🟢
Record of a meeting, prepared immediately after & circulated. Includes date/time/venue, chairperson, participants, conclusions, and responsible Ministry/Dept for follow-up action.

Drafting of Communication (Ch. VII) 🔴प्रारूपण नियम
Rule 41-42: Drafting Rules
  • No draft needed for straightforward cases / standard-form communications
  • Draft must be clear & concise; always refer to number/date of last communication
  • Two+ communications from same file, same date → separate serial no. before year, e.g. A-11011/5(I)/2001-Est., A-11011/5(II)/2001-Est.
  • Signing officer's name, designation, phone, fax, email must be indicated
Rule 43: Authentication of Orders (MHA)

Orders in President's name → signed by officer of Under Secretary rank or above (regular/ex-officio secretariat status), per Authentication (Orders & Other Instruments) Rules, 2002.

Rule 44: Addressing by Name

Normally NOT addressed by name (except classified/D.O.) unless matter needs personal attention due to special nature, urgency, or prior discussion.

  • Unique e-File Number: auto-generated; traceable by file no., date, catchwords

Records Management (Ch. IX) 🟡अभिलेख प्रबंधन
  • Files recorded after action completed, per Record Retention Schedule (RRS) & NAI norms
  • Precedent Book — updated by every section for important rulings
  • Accounts records → follow Appendix 13, GFR
  • Establishment/personnel/housekeeping → common RRS (DARPG website)
  • Recorded file requisition → prescribed form under Public Records Act, 1993
  • Security-classified records → NOT transferred to NAI (Sec. 10, Public Records Act 1993)
10
Security of Official Information (Ch. X) 🟡सुरक्षा एवं गोपनीयता
  • RTI Act 2005 — officials must communicate accurate info that can be disclosed
  • Classified files → reviewed once in 5 years for declassification
  • Carrying confidential papers outside office: Section/Desk Officer level allowed; Secret papers: Under Secretary level, with written JS-level authorization
  • Press info → normally through Press Information Bureau (PIB) only
  • "For Official Use Only" — used only if disclosure not in public interest
11
Checks on Delays (Ch. XI) 🔴विलंब पर नियंत्रण
⏱️ Golden Rule: No official shall keep a case pending for more than 7 working days unless a higher limit is prescribed by departmental instructions. If exceeded → explanation must be recorded on the note portion.

Applies to: Parliamentary Assurances, Parliament Questions, RTI Applications, MP/VIP References, CAT/Court Cases — each maintained as separate records; E-Governance tools used for tracking.

Miscellaneous (Ch. XII) 🟢विविध
  • Citizen's/Client's Charter — mandatory for all Ministries/Depts, displayed on website + Notice Board of IFC
  • 7 components: Vision/Mission, Business details, Citizens/Clients, Service standards, Grievance mechanism, Expectations, Compensation commitment
  • Websites must follow GIGW guidelines; nodal officer to be appointed

🧠 Memory Tricks / Mnemonicsयाद रखने की तरकीबें
"LDOOR ONPEM"
11 Forms of Communication → Letter, D.O. Letter, Office Memorandum, Office Order, Resolution, Order, Notification, Press Note, Endorsement, Minutes, (I.D. Note)
"L-P-T" for Urgency
Low urgency levels rise: Priority → Top Priority → Immediate (most urgent). Remember: "Immediate" beats all!
"RED = Lok Sabha" (Red carpet of Lok Sabha)
Lok Sabha file = RED cover; Rajya Sabha file = GREEN cover (like traffic: Lok Sabha "stops" everything = red; Rajya Sabha = go/green, the "elder" calm house)
"Left Hand Assists, Right Hand Rules"
Dealing Hand (assistant/junior) signs on LEFT; Officer (senior/rules) signs on RIGHT of the note.
"7 Days = 1 Week Rule"
No case pending beyond 7 working days without explanation — think of it as "1 working week" limit.

❓ Probable Exam Questionsसंभावित प्रश्न
Q1. What is the maximum number of levels through which a file can pass for a decision?
Ans: 4 levels (Rule 14)
Q2. Which form of communication is written in the first person and used for personal attention?
Ans: D.O. (Demi-Official) Letter
Q3. What was the earlier name of the Inter-Departmental Note?
Ans: U.O. Note (before 7th Edition CSMOP)
Q4. What colour is used for Lok Sabha and Rajya Sabha file covers respectively?
Ans: Red and Green
Q5. Within how many days should a case not be kept pending, as a general rule?
Ans: 7 working days (Ch. XI, Rule 60)
Q6. On which side does the Dealing Hand sign a note, and on which side does the Officer sign?
Ans: Dealing Hand — Left; Officer — Right
Q7. When is a file marked "Volume I", "Volume II"?
Ans: When notes + correspondence exceed 150-200 pages
Q8. Which form is used for Gazette publication of statutory rules and appointments of gazetted officers?
Ans: Notification
Q9. What is the minimum rank for authenticating orders issued in the name of the President?
Ans: Under Secretary or above
Q10. How often must a classified file be reviewed for declassification?
Ans: Once in 5 years
Q11. What is "PUC" and what is "FR" in file referencing?
Ans: PUC = Paper Under Consideration; FR = Fresh Receipt
Q12. Which Act governs requisitioning of records, and in which year?
Ans: Public Records Act, 1993
