📮 LDCE IP Exam Portal
A self-hosted, mobile-first exam preparation portal for the Inspector Posts (IP) Departmental Examination, India Post. Built as a personal study companion covering Paper I, Paper II, and Paper III syllabus in a structured, offline-friendly format.
🔗 Live Portal: lifedistructor-dev.github.io/LDCE-IP-EXAM-PORTAL-
✨ Features
📚 Structured by Paper — Content organized into Paper I (Acts, Rules, Manuals), Paper II (Noting/Drafting/Charge Sheet), and Paper III (Constitution, BNSS, MCQ Banks)
🌗 Dark Mode — Eye-friendly reading for late-night study sessions
🔊 Text-to-Speech — Listen to content on the go using the built-in TTS widget
📴 Offline Support — PWA with service worker caching, so content is accessible without internet
🔖 Bookmarks & Progress Tracking — Mark topics as studied and save important sections for quick revision
🔍 Instant Search — Search across Acts, Rules, and PLI content directly from the home screen
🎨 Glassmorphism UI — Clean, modern interface built for quick mobile reading
📖 Content Coverage
Paper I
CCS Rules (Pension, CCA, Conduct, Gratuity, NPS), Financial Handbooks (Vol. I & II), GFR/DFPR, Postal Manuals (Vol. II–VIII), POSB Manuals & CBS, RTI Act & Rules, Procurement Manual, Post Office Act 2023, GDS Conduct Rules, IT Act, PMLA, and more.
Paper II
Noting, Drafting & Charge Sheet practice sets — including CCS Conduct/CCA Rules, CSMOP formats, and Top-10 Most Probable Question Banks with model answers and exact rule/regulation references.
Paper III
Constitution of India, BNSS 2023, CAT Act, Revenue Recovery Act, IT Modernization 2.0, IPPB Basics, Finacle CBS Commands, and 250+ question MCQ banks in Gujarati/Hindi.
🛠️ Tech Stack
Pure HTML, CSS, and JavaScript — no build tools, no frameworks
Progressive Web App (PWA) with offline-first service worker
Web Speech API for text-to-speech
Hosted free on GitHub Pages
Built and maintained entirely on mobile using the Acode editor
📂 Structure
Each topic is a self-contained HTML file linked from the central index.html dashboard, grouped by paper and sub-category with collapsible accordion cards for easy navigation.
